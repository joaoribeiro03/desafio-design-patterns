# Padrões de Projeto em Java

Este repositório contém exemplos de código em Java puro que demonstram a implementação de três padrões de projeto comuns: Singleton, Facade e Strategy. Além disso, há um desafio relacionado ao padrão Strategy no pacote `edu.joaolucas.gof.desafio.strategy`. 

No desafio, foi criada uma interface `Sorter` que padroniza as diferentes estratégias de um algoritmo, juntamente com as classes `InsertionSorter` e `SelectionSorter` que implementam de forma particular uma estratégia.

## Conteúdo do Repositório

- **Singleton**: Exemplo de implementação do padrão Singleton.
- **Facade**: Exemplo de implementação do padrão Facade.
- **Strategy**: Exemplo de implementação do padrão Strategy.
- **Desafio Strategy**: Desafio relacionado ao padrão Strategy, com uma interface `Sorter` e implementações específicas.
